# Claude Code

Claude Code 是一种偏本地交互式的 AI 代理/命令行工具，常被用于直接读取和修改本地文件、驱动 Obsidian vault 工作流，或把对话整理成 wiki 页面。当前知识库中，它主要出现在围绕 Obsidian 与 LLM Wiki 的使用场景里。 ^[raw/articles/claude-obsidian-second-brain-2026-04-14.md] ^[raw/articles/obsidian-basics-git-sync-2026-06-30.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]

## 常见用法

- 在 vault 中生成、编辑和整理 Markdown 页面。 ^[raw/articles/obsidian-basics-git-sync-2026-06-30.md]
- 与 Obsidian 工作流/插件结合，执行 ingest、research、canvas 等知识库操作。 ^[raw/articles/claude-obsidian-second-brain-2026-04-14.md]
- 在入门讨论中，经常与 [[hermes-agent]] 一起被比较：都能做事，但生态定位和交互方式不同。 ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]
