# Obsidian

Obsidian 是一个本地 Markdown 笔记工具，适合作为 [[llm-wiki]] 的前端：它原生支持双链、图谱视图、附件管理和社区插件。

## 在 LLM Wiki 中的作用

- 用普通 Markdown 文件承载长期知识。
- 通过 Obsidian 双链语法形成知识图谱。
- Graph View 可发现孤岛页面、缺失页面和知识盲区。
- Web Clipper 可把网页资料保存为 Markdown，作为 [[source-ingestion-workflow]] 的来源入口。
- Git 插件可为 AI 批量修改提供版本回滚能力。
