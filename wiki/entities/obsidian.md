# Obsidian

Obsidian 是一个本地 Markdown 笔记工具，也可以被组织成 AI 的长期工作空间：资料、模板、历史结果、索引和交叉链接都能保存在同一个 vault 中。它不只是记笔记软件，还可以作为 [[llm-wiki]]、自动化流程和第二大脑的承载层。 ^[raw/articles/karpathy-llm-wiki-2026-04-13.md] ^[raw/articles/obsidian-ai-workflows-2026-04-22.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]

## 在 LLM Wiki 中的作用

- 用普通 Markdown 文件承载长期知识。
- 通过 Obsidian 双链语法形成知识图谱。
- Graph View 可发现孤岛页面、缺失页面和知识盲区。 ^[raw/articles/karpathy-llm-wiki-2026-04-13.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]
- Web Clipper、Readwise 类入口和手工导入都可作为 [[source-ingestion-workflow]] 的资料来源。 ^[raw/articles/karpathy-llm-wiki-2026-04-13.md] ^[raw/articles/obsidian-ai-workflows-2026-04-22.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]
- 它还可以保存 AI 工作流所需的模板、配置和中间结果，让不同 AI 工具围绕同一套本地文件协作。 ^[raw/articles/obsidian-ai-workflows-2026-04-22.md]

## 基础实践要点

- 新手路径通常从安装、创建 vault、掌握 Markdown 基础与双向链接开始。 ^[raw/articles/obsidian-basics-git-sync-2026-06-30.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]
- 不少实践会结合 Git 或插件完成免费同步与版本回滚。 ^[raw/articles/obsidian-basics-git-sync-2026-06-30.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]
- 对非技术用户来说，Obsidian 的价值在于“文件是自己的、结构是自己的、AI 工具可替换”。 ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md] ^[raw/articles/obsidian-ai-workflows-2026-04-22.md]
