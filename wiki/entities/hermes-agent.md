# Hermes Agent

Hermes Agent 是一个面向终端和多平台消息网关的 AI Agent 系统，强调工具调用、跨会话记忆、可复用技能和多渠道接入。多份资料都把它描述为“能随着使用变得更顺手”的代理工具，可用于知识整理、网页抓取、自动总结与多渠道分发。 ^[raw/articles/hermes-agent-guide-2026-04-15.md] ^[raw/articles/hermes-agent-orange-book-2026-06-09.md]

## 常见定位

- 作为通用型 Agent 执行层，适合和 [[obsidian]] 或 [[llm-wiki]] 配合。 ^[raw/articles/hermes-agent-guide-2026-04-15.md] ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]
- 相比把重点放在单一编辑器工作流的方案，Hermes 更强调多工具、多平台、长期记忆与技能沉淀。 ^[raw/articles/hermes-agent-orange-book-2026-06-09.md] ^[raw/articles/hermes-agent-guide-2026-04-15.md]
- 在入门语境里，它常被拿来与 [[claude-code]] 对比：前者更像通用 Agent 平台，后者更像强交互的本地编码/文件代理。 ^[raw/articles/youtube-obsidian-second-brain-tutorial-2026-06-30.md]

## 在当前知识库里的意义

- 可作为执行 [[obsidian-ai-workflows]] 的代理层。
- 适合把知识整理、网页抓取、自动总结、消息分发等流程串起来。
- 与 [[llm-wiki]] 的关系是：它不是知识库本身，而是可以维护知识库和运行工作流的代理工具。
