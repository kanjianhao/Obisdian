# Hermes Agent

Hermes Agent 是一个面向终端和多平台消息网关的 AI Agent 系统，强调工具调用、跨会话记忆、可复用技能和多渠道接入。现有资料把它描述为既可在本地 CLI 中使用，也可接入 Telegram、飞书、钉钉等平台。由于当前页面主要来自单一二手教程，具体命令和支持范围在实操时应再核对官方文档。 ^[raw/articles/hermes-agent-guide-2026-04-15.md]

## 在当前知识库里的意义

- 可作为执行 [[obsidian-ai-workflows]] 的代理层。
- 适合把知识整理、网页抓取、自动总结、消息分发等流程串起来。 ^[raw/articles/hermes-agent-guide-2026-04-15.md]
- 与 [[llm-wiki]] 的关系是：它不是知识库本身，而是可以维护知识库和运行工作流的代理工具。
